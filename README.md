# Quake 2 - Instituto Alfonso X (1999)

3 mapas custom de Quake II creados en 1999 con texturas originales, jugables directamente en el navegador.

**Creado por [Javi](https://x.com/javilopen) y [Emilio](https://x.com/emailnicolas) en 1999.**

## Jugar online

**[https://emilionicolas.github.io/quake2-alfonso-x-1999/](https://emilionicolas.github.io/quake2-alfonso-x-1999/)**

La primera vez descarga automaticamente la demo shareware de Quake II (~37MB) para los assets base del juego.

## Mapas

| Mapa | Descripcion |
|------|-------------|
| **ALFONSOX** | El Instituto Alfonso X - mapa principal |
| **ALF2** | Expansion del instituto |
| **DISCO** | La discoteca |
| **FINAL** | Pantalla de victoria |

Los 4 mapas estan encadenados: ALFONSOX -> ALF2 -> DISCO -> FINAL

## Controles

| Tecla | Accion |
|-------|--------|
| WASD | Movimiento |
| Mouse | Mirar |
| Click izquierdo | Disparar |
| Space | Saltar |
| Shift izquierdo | Agacharse |
| 1-9 | Cambiar arma |
| E | Usar item |
| Scroll | Cambiar arma |

## Contenido custom

- 4 mapas BSP
- 31 texturas WAL originales
- Sonido de trigger personalizado
