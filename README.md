# Quake 2 - Instituto Alfonso X (1999)

Salva al Instituto Alfonso X de la invasion alienigena. Recorre sus pasillos, clases y hasta la discoteca en estos mapas custom de Quake II creados en 1999 con texturas originales.

**Creado por [Javi](https://x.com/javilopen) y [Emilio](https://x.com/emailnicolas) en 1999.**

## Jugar online

**[https://emilionicolas.github.io/quake2-alfonso-x-1999/](https://emilionicolas.github.io/quake2-alfonso-x-1999/)**

La primera vez descarga automaticamente la demo shareware de Quake II (~37MB) para los assets base del juego.

## Controles

| Tecla | Accion |
|-------|--------|
| WASD | Movimiento |
| Mouse | Mirar |
| Click izquierdo | Disparar |
| Space | Saltar |
| Shift izquierdo | Agacharse |
| 1-9 | Cambiar arma |
| E | Usar item |
| Scroll | Cambiar arma |
